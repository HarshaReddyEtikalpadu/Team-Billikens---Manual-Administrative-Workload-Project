Lessons Learned & Team Reflection

Team Dynamics

Five-member team worked in agile sprints with GitHub collaboration. Roles were distributed to balance technical, design, and documentation needs.

Major Decisions

  • Synthetic vs Real Data: Chose synthetic data to protect privacy while enabling realistic testing.
  • Technology Stack: React chosen for portfolio value and rich UX; Python for data generation.
  • Feature Prioritization: Focused on core analytics and visualization rather than full ML predictions.

Key Takeaways

  • Technical: Hands-on experience with React, charting, and data engineering principles.
  • Process: Iterative prototyping and early stakeholder feedback are essential.
  • Professional: This project resulted in portfolio-ready artifacts useful for interviews and demos.
