Agile Collaboration

Five-member team worked in sprints with GitHub workflows, balancing technical, design, and documentation responsibilities.

🔧

Major Decisions

Synthetic vs Real Data: Chose synthetic data to protect privacy while enabling realistic testing.

Technology Stack: React for rich UX; Python for data generation and shaping.

Feature Prioritization: Focused on analytics and visualization before advanced ML predictions.

🎯

Key Takeaways

Technical: React, charting, and data engineering practice.

Process: Iterative prototyping with early stakeholder feedback.

Professional: Portfolio-ready artifacts for demos and interviews.
